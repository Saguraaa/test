#### 思路
##### 1 .加载函数的设计
检索配置文件每一行，并检查配对<T>类型的每个属性
忽略开头为 #和 //以及空行
根据=分割字符串，前面的为属性名，后面的为数值
遍历之前检查的属性名并与分割出来的属性名一一对应
若配对上则对数值字符串进行类型转换

##### 2 . 保存函数的设计
先检查要保存类的每一个属性和特性
直接在新建文件中以"属性"="数值"的形式写出
